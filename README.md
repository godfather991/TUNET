### 使用

- 下载代码：

  ```bash
  git clone https://github.com/godfather991/TUNET.git
  cd TUNET
  ```

- 代码中修改：

  1. 选择无线/有线连接类型
  2. 输入自己的用户名/密码

- 命令行启动：

  ```bash
  python login.py
  ```
